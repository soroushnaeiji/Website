React Tutorial

React is a JavaScript library for building user interface. It allows you to create resuable UI components that can manage their own state.

In the examples below you woll see some examples of a react components

Basic React Component

A react component is a resuable piece of the UI. Components can be defined as functions or classes. here is a simple example of a functional React Component:

            < script type="text/babel">
            function Counter(){
                const[count, setCount] =  React.useState(0);

                return(
                    <div>
                        <p> You clicked {count} times </p>
                        <button onClick = {() = > setCount(count +1)}>
                        click me!
                        <button>
                    </div>

                );
            }

the code above shows the react code where you can keep track of the number of times that the button has been cliced on
