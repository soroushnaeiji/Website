Welcome to the HTML and CSS tutorial page

HTML also known as HyperText Markup Language is the standard language dor creating web pages. It describes the strucutre of a webapge using a series of elements. Tags are also another components that are being used in this language.

CSS which is known as Cascading Style Sheets is used to style and layout the webpage. It allows you to apply style to HTML elements. The styles can be colors, fonts, and spacing

Below you will see the most common tags that are being used when developing webpage. Below each tag, you will see and example if applicable and a definition of what the tag is and where it would be used. 

                <!DOCTYPE html>
                
This line of code defines the document as a HTML document

                <head></head>
                
This line defines the head of the website.  It is also knows as the upper part where you have the tile of the website.  You can also define values such as 
                    styling sheet, encoded reading, links, and different font designs from external sources. 
                

                <title>
                
This tag changes the name on the actual tab in the broswer


                <img> 
                
                
This tag helps with adding images to the website!  You need to define a source for it where you will add the path of the image you want to add to your website
